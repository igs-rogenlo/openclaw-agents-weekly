💬 自然語言群組通知（Telegram）升級

將 Agent 拉進 Telegram 群組後，用自然語言設定提醒規則：定時或收到特定訊息時 tag 指定成員，該成員回覆後自動取消提醒
💡 範例：定時提醒某人直到回覆
👤 PM：「每天下午 3 點提醒 @小明 回報進度，他回覆就取消」
🤖 Agent：已設定 ✅ 每天 15:00 會 tag 小明，他回覆後自動停止
💡 範例：收到訊息觸發提醒
👤 主管：「如果群組收到『版本釋出』，就 tag @阿華 開始測試」
🤖 Agent：已設定 ✅ 偵測到「版本釋出」時會自動 tag 阿華
💡 範例：每隔一段時間催繳
👤 行政：「每 2 小時提醒 @志偉 交週報，他丟了週報就不用再提醒」
🤖 Agent：已設定 ✅ 每 2 小時 tag 志偉，收到他的週報後自動停止

🧵 Discord 一人多 Chat 升級

同一人可以對同一個 Agent 同時開多個對話，各自獨立不混淆
💡 情境
Chat 1 — 跟機率 Agent 討論「A 遊戲的中獎率設定」
Chat 2 — 同時跟機率 Agent 問「B 遊戲的免費旋轉機率」
兩個對話各自記住上下文，不會搞混
